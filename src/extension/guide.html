1. How to Use the Extension

Auto-Translate & Inject

If you are watching video streams with English subtitles, the extension can translate and overlay them instantly:

  1. Open the stream page (e.g., Anistream) and start playing the video.
  2. Open the extension popup. If supported player subtitles are found, you will see a green "English Subtitles Detected!" notice.
  3. Click the "Auto Translate & Inject" button. The translation will execute, and the target subtitles will be overlayed onto the video player.

Manual Subtitle Injection

You can also translate and inject your own local subtitle files:

  1. Drag & drop a subtitle file (.srt, .vtt, .ass, or .lrc) into the upload zone.
  2. Select the Source Lang and Target Lang (e.g. Khmer).
  3. Click "Translate & Inject" to translate the subtitles using Gemini AI and inject them into the player.
  4. Or, click "Inject Only" to overlay the subtitle file onto the video immediately without translating.
  5. After translation completes, you can click "Download Translated" to save the file to your disk.

2. Supported Streaming Sites

The extension contains built-in injectors that auto-detect and sync subtitles on these platforms:

  • Anistream
  • Khanime
  • KHFullHD

3. AI Settings & Configurations Explained

Gemini API Key Required for AI

Your personal API key from Google AI Studio. Translations run locally and securely directly from your browser to Google Gemini.

Select Model Configurable

Choose which Gemini model to use. gemini-3.5-flash is recommended for general translation as it is extremely fast and cost-efficient.

Context-Aware Translation Toggle

When enabled, the translator will send surrounding subtitle lines (before and after the current block) as context to the LLM. This yields much more natural dialogue, correct pronouns, and consistent narrative flow.

Concurrency Slider

Determines how many lines are sent in parallel. Higher values accelerate translation but increase the risk of hitting Gemini API limits (Error 429).

Delay (ms) Slider

The delay between requests. If your translations are failing due to rate limit errors, increase this delay to space out API requests.

18+ / Mature Content Toggle

Allows the LLM to translate slang, swearing, and adult/violent dialogue accurately without censoring or triggering standard safety blocks.

Enable Caching Toggle

Saves all translated lines to a local cache (IndexedDB). If you re-translate the same subtitle file or replay a video, the translations load instantly without hitting the Gemini API again, saving you cost and time.

1. 插件使用方法

自动翻译并注入

如果您正在观看含有英文字幕的视频流媒体，插件可以实时翻译并显示：

  1. 打开播放网页（如 Anistream）并开始播放视频。
  2. 打开插件弹出窗口。若成功检测到字幕，将显示绿色的 "检测到英文字幕！" 提示。
  3. 点击 "自动翻译并注入"。翻译将启动，且译文字幕将实时注入并同步覆盖到视频播放器上。

手动字幕注入

您也可以上传并注入本地字幕文件：

  1. 将字幕文件（支持 .srt, .vtt, .ass, .lrc）拖拽到上传框中。
  2. 选择 源语言 和 目标语言（如高棉语或中文）。
  3. 点击 "翻译并注入" 使用 Gemini AI 进行翻译并插入到播放器中。
  4. 或者点击 "仅注入" 直接在当前视频中加载此字幕（无需翻译）。
  5. 翻译成功后，您可以点击 "下载已翻译字幕" 以将文件保存到电脑。

2. 支持的流媒体网站

插件内置专用的播放器探测器，支持自动检测并在以下网站注入字幕：

  • Anistream
  • Khanime
  • KHFullHD

3. AI 设置说明

Gemini API Key 必填项

您的个人 Google AI Studio API 密钥。翻译工作完全在您本地浏览器发起并直达 Google，安全无风险。

选择模型 可配置

选择要调用的 Gemini 模型。推荐使用 gemini-3.5-flash，翻译速度极快，性价比极高。

上下文关联翻译 开关

启用后，翻译时会一并发送当前行前后的字幕作为上下文参考，使代词人称更准确，对白更流畅连贯。

并发数 (Concurrency) 开关

设定并行发送的请求数量。并发越高速度越快，但更容易触发 API 429 报错限制。

延迟时间 (Delay) 开关

两次 API 请求之间的间隔时间（毫秒）。如果您的翻译频繁因频率限制报错，请调大此值以缓和请求频率。

18+ / 成人及暴力内容 开关

允许 LLM 准确地翻译脏话、俚语以及暴力成人对话，不进行任何人工过滤或自我审查。

启用本地缓存 开关

开启缓存（使用 IndexedDB）后，翻译结果会保存在您本地。如果再次翻译相同文件或重播视频，将秒速加载结果，且完全不再消耗任何 API 额度。
